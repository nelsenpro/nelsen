Proyek Utama

Jelajahi beberapa proyek open source yang telah saya kembangkan dan kontribusikan.

Frame.css

**Frame.css** adalah proyek open-source yang menyediakan beragam **library CSS** untuk mendesain aplikasi web dan website dengan mudah dan stylish.

Selengkapnya...

Fonia.css

**Fonia.css** adalah kumpulan pustaka CSS kecil dan ringan yang digunakan untuk mewarnai font/teks, latar belakang, perataan teks, dan jenis font untuk desain web. Ini adalah proyek lain dari saya, seorang pengembang **CSS Framework Indonesia**.

Selengkapnya...

CodeCanvas.js

**cc.js** atau **CodeCanvas.js** adalah pustaka **JavaScript ringan** yang memungkinkan Anda menjalankan JavaScript dan menerapkan gaya secara dinamis dalam dokumen HTML menggunakan tag "cc".

Selengkapnya...

MyQR

**MyQR** adalah generator kode QR yang dibangun dengan **HTML, CSS, dan JavaScript murni**, memungkinkan Anda membuat kode QR dari berbagai jenis data dengan mudah.
